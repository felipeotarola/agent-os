# Sladdis QA Vertical: SEO/Content

Use this planned strategy when Felipe asks for SEO, metadata, content QA, copy review, search visibility, or share preview testing.

## Goal

Check whether people and search systems can understand the page topic, offer, proof, and next action.

## Required Checks

- Page title and meta description.
- Heading hierarchy and topic clarity.
- Link labels and internal navigation.
- Social sharing preview readiness.
- Search intent fit.
- Trust, proof, and objection-handling copy.
- Thin, duplicate, outdated, or confusing content.

## Report Rules

- Use the future `seo-content-report` template.
- Avoid promising ranking outcomes.
- Keep recommendations concrete and tied to page sections.

## Output Shape

Return a `QaReport` with `vertical: "seo-content"` once the template is implemented.
