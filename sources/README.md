# Agent OS Sources

This directory holds local source bundles: raw or lightly summarized evidence that should stay separate from interpreted context.

Use `docs/SOURCES_LAYER.md` for the rules. Keep this directory small. Most sources should live in the Knowledge Inbox, existing task descriptions, repo links, or external systems; use files here only when a local artifact needs a stable citation.

## File Shape

```markdown
# Source: short title

- Source ID: `src:YYYY-MM-DD-short-topic`
- Date: YYYY-MM-DD
- Owner: cai
- Sensitivity: public | internal-summary | approval-gated
- Related task: `<task-id or none>`

## Raw Or Summary

Short evidence summary or a link list. Do not paste secrets or unapproved private data.

## Used By

- `<doc, decision, task, or agent output>`
```
